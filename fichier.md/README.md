# Site Portfolio — Ophélie, Photographe

Site web vitrine réalisé pour une cliente photographe, **Ophélie**, afin de mettre en valeur son travail et son univers artistique.

## À propos du projet

Ce site a été conçu et développé en tant que projet freelance. L'objectif était de créer une expérience visuelle élégante et immersive, mettant en avant les shootings photos d'Ophélie.

## Fonctionnalités

- Galerie photo avec mise en page soignée
- Design responsive (mobile, tablette, desktop)
- Navigation fluide et intuitive
- Présentation des différentes séances photo

## Technologies utilisées

- HTML5
- CSS3
- JavaScript (Vanilla)
